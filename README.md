# ia-codyssey

AI 코디세이 과정에서 수행한 미션과 프로젝트를 체계적으로 관리하는 저장소입니다.

---

## 📁 저장소 구조

```text
ia-codyssey
│
├── README.md
│
├── 미션03_노코드_기반_워크플로우_설계
│
├── 미션04_...
│
└── ...
```

---

## 📌 진행 중인 미션

| 미션 | 주제 | 상태 |
|------|------|------|
| 미션03 | 노코드 기반 워크플로우 설계 | 🚧 진행 중 |

---

## 📖 저장소 운영 규칙

각 미션은 아래와 같은 구조를 따릅니다.

```text
미션XX
│
├── README.md
├── 프로젝트1_보고서.md
├── 프로젝트2_보고서.md
└── 자료
    ├── 프로젝트1
    └── 프로젝트2
```

새로운 미션은 동일한 구조로 순차적으로 추가됩니다.
