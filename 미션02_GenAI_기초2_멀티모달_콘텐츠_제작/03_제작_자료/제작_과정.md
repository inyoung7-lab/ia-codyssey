## 1. 제작 기준

| 항목 | 기준 |
|---|---|
| 브랜드 | EchoLens |
| 제품 | EchoLens AI Smart Glasses |
| 광고 길이 | 10초 이내 |
| Scene | Scene 1~5, 각 편집 구간 2초 |
| 화면 비율 | 16:9 |
| 비주얼 방향 | Apple의 프리미엄 감성, GoPro의 역동적인 여행, 영화 같은 시네마틱 연출 |

## 2. Image 생성

1. 브랜드 아이덴티티와 Style Guide에서 캐릭터, 제품 디자인, 색감과 촬영 스타일을 고정했다.
2. Master Prompt와 Negative Prompt를 공통 기준으로 정리했다.
3. Scene 1~5의 목적과 화면 구성에 맞춰 영어 이미지 Prompt를 작성했다.
4. Scene 1은 POV 몰입감에 집중하고, EchoLens 제품 노출은 Scene 3으로 분리했다.

관련 문서: [Image Prompt Guide](Prompts/image_prompts.md)

현재 `03_제작_자료/Images/`에는 이미지 원본 파일이 없어, 문서에 적힌 이미지 파일명은 Prompt 기록상 예정 파일명으로 유지한다.

## 3. Video 생성

1. Scene별 정지 이미지를 기준으로 Kling AI용 영상 Prompt를 작성했다.
2. 각 Scene에 카메라 움직임, 전환 방식, 16:9 비율과 2초 편집 구간을 지정했다.
3. 여행 장면은 역동적으로, 제품과 브랜드 엔딩은 안정적인 구도로 설계했다.
4. Kling 크레딧 제약에 대응해 Scene 수를 조정하고 Scene 3을 최종 광고의 오프닝으로 재구성했다.
5. Scene 5는 Kling AI 영상을 추가 생성하지 않고 정지 이미지와 Fade 효과로 처리했다.

관련 문서: [Video Prompt Guide](Prompts/video_prompts.md)

Kling AI로 제작한 주요 Scene 영상은 다음과 같다.

| Scene | 실제 파일명 | 길이 |
|---|---|---|
| Scene 2 | `scene02_motion.mp4` | 약 3초 |
| Scene 3 | `scene03_motion.mp4` | 약 3초 |
| Scene 4 | `scene04_motion.mp4` | 약 3초 |

### Pika 비교 재제작

동일 Scene을 다른 생성형 AI에서도 구현하기 위해 Scene 3을 Pika의 Text to Video 방식으로 추가 제작했다.

| 항목 | 실제 결과 |
|---|---|
| 파일 | `03_제작_자료/Videos/scene03_pika_remake.mp4` |
| 규격 | 5초, 784×470, 30fps |
| 유지 요소 | 해안 절벽, 셀피 구도, AI 스마트글래스 콘셉트 |
| 차이 | 남성에서 여성 인물로 변화, 슬림한 안경에서 넓은 바이저형 디자인으로 변화 |
| 장점 | 자연스러운 셀피 움직임 |
| 생성 차이 | 인물 및 제품 디자인이 Kling 원본과 다르게 표현됨 |

### Hedra Lip-sync 추가 제작

1. Hedra에 회원가입했다.
2. 기존 Scene 3 대표 이미지를 업로드했다.
3. `Travel Free. Record Smart.` 음성을 생성했다.
4. Talking Avatar를 생성했다.
5. Lip-sync 영상 `01_스토리보드_기획/Bonus/scene03_hedra_lipsync.mp4`를 생성했다.
6. 결과는 Scene 3 기반 Lip-sync 적용 사례로 정리했다.

Hedra 파일은 3.23초, 1280×720, H.264 영상과 AAC 오디오로 정리했다.

스토리보드의 2초 표기는 최종 편집 구간 기준이며, 저장된 Kling 원본 클립의 확인 길이와 구분한다.

## 4. Audio 생성

1. Suno AI용 광고 BGM Prompt를 작성하고 16:9 최종 광고용 BGM을 생성했다.
2. 생성한 BGM은 `03_제작_자료/Audio/echolens_ai_bgm_suno.mp3`에 저장했다.
3. Suno AI BGM을 적용한 16:9 최종 광고는 `02_광고_영상/EchoLens_AI_Travel_Commercial_10s.mp4`로 Export했다.
4. 9:16 세로 버전은 기존 CapCut 라이브러리 BGM `Cinematic Epic Inspiration`을 유지했다.
5. 배경음악은 시네마틱하고 프리미엄한 광고 분위기와 여행의 확장감을 표현하는 용도로 적용했다.
6. CapCut에서 10초 광고 길이에 맞춰 BGM을 배치하고 오디오 길이와 레벨을 조정했다.
7. Scene 1~5의 내레이션 문구와 시간 구간은 스토리보드 기획 단계 카피로 정리했다.

관련 문서: [Audio Prompt Guide](Prompts/audio_prompts.md)

Suno AI 생성 BGM은 16:9 최종 광고에 적용했고, CapCut 라이브러리 BGM은 9:16 영상에 유지했다.

## 5. CapCut 편집

CapCut은 생성 결과물을 하나의 광고 흐름으로 통합하는 편집 도구로 사용했다. 생성 결과물을 타임라인에 배치하고 컷 편집, 전환, 색보정, 오디오 길이 조정, 오디오 레벨 조정, Fade In/Out, 브랜드 엔딩과 CTA를 구성했다. Scene 5는 정지 이미지와 Fade 효과로 처리했다.

### 9:16 플랫폼 버전 제작

1. CapCut에서 16:9 최종 광고를 기준으로 9:16 세로 프로젝트를 구성했다.
2. 세로 화면에서 주요 인물, 여행 장면, AI UI와 브랜드 엔딩이 보이도록 Scene별 화면을 확대했다.
3. 각 Scene의 핵심 요소가 세로 프레임 안에 유지되도록 화면 위치를 조정했다.
4. `02_광고_영상/EchoLens_AI_Travel_Commercial_9x16.mp4`로 Export했다.

9:16 버전은 기존 CapCut 라이브러리 BGM `Cinematic Epic Inspiration`을 유지했다. MP4, 약 10.01초, 1080×1920, 30fps, H.264, AAC stereo이며 Shorts, Reels, TikTok 세로 화면 활용을 목적으로 한다.

CapCut 편집 과정은 최종 Export 결과와 제작 단계 기록을 기준으로 정리했다.

## 6. Export

Suno AI BGM을 적용한 16:9 최종 광고는 `02_광고_영상/EchoLens_AI_Travel_Commercial_10s.mp4`로 Export했다. 실제 파일은 MP4, 약 10.01초, 1920×1080, 30fps, H.264, AAC stereo로 정리했다.

플랫폼별 화면비 버전은 기존 CapCut 라이브러리 BGM `Cinematic Epic Inspiration`을 유지해 `02_광고_영상/EchoLens_AI_Travel_Commercial_9x16.mp4`로 Export했다. 실제 파일은 MP4, 약 10.01초, 1080×1920, 30fps, H.264, AAC stereo로 정리했다.

### 최종 편집 타임라인

초기 스토리보드는 Scene 1 → Scene 2 → Scene 3 → Scene 4 → Scene 5 순서로 기획했다. 최종 편집에서는 Kling AI 크레딧 제약과 실제 결과에 따라 Scene 3 → Scene 2 → Scene 4 → Scene 5 순서로 재구성했다.

| 시간 | 실제 반영 내용 |
|---|---|
| 0~3초 | Scene 3: 해안 절벽을 배경으로 EchoLens를 착용한 여행 크리에이터 |
| 3~6초 | Scene 2: 바다·도시·산의 3분할 1인칭 POV 여행 장면 |
| 6~8초 | Scene 4: 여행 클립을 정리하는 EchoLens AI 편집 UI |
| 8~10초 | Scene 5: EchoLens 로고, 슬로건, CTA 브랜드 엔딩 |

Scene 1은 여행 시작을 보여주는 초기 콘셉트로 설계했고, 최종 편집에서는 제품의 첫인상을 강화하기 위해 Scene 3을 오프닝으로 구성했다. Scene 2~4에는 Kling AI 결과물의 `KlingAI 3.0` 워터마크가 표시된다.

## 7. AI 도구 선정

| 도구 | 선택 이유 | 장점 | 보완점 |
|---|---|---|---|
| ChatGPT 이미지 생성 | 사실적인 여행·제품 이미지와 일관된 스타일 생성 | 공통 Prompt와 Scene별 변형을 함께 관리할 수 있음 | 캐릭터와 제품 일관성을 반복 Prompt로 관리해야 함 |
| Kling AI | 정지 이미지를 자연스러운 여행 영상으로 변환 | 카메라 이동과 장면 전환을 짧은 모션으로 구현할 수 있음 | 크레딧 제약으로 생성 범위를 조정해야 했음 |
| Pika | Scene 3 재제작과 생성 결과 비교 | Text to Video에서 자연스러운 셀피 움직임 구현 | 인물 성별과 안경 디자인이 Kling 원본과 다르게 표현됨 |
| Hedra | Scene 3 Talking Avatar Lip-sync 추가 제작 | 기존 이미지를 음성에 맞춰 Lip-sync 영상으로 구현 | Scene 3 기반 추가 제작 결과로 정리 |
| Suno AI | 16:9 최종 광고용 BGM 생성 | 프롬프트 기반으로 광고 분위기에 맞는 시네마틱 BGM을 직접 생성할 수 있음 | 16:9 최종본의 BGM으로 활용 |
| CapCut 라이브러리 BGM | 9:16 세로 버전의 기존 음악 유지 | `Cinematic Epic Inspiration`을 편집 타임라인에 적용 가능 | 9:16 세로 버전의 라이브러리 음악으로 활용 |
| CapCut | BGM 적용, 생성 결과물 통합과 최종 Export | 컷 편집, 전환, 색보정, 오디오 레벨, 브랜드 엔딩을 한 타임라인에서 편집 가능 | 생성 결과물을 광고 흐름으로 통합하는 편집 도구로 활용 |

| 유료 스톡 영상 | 직접 설계한 Prompt 기반 생성 결과를 핵심 시각 요소로 사용했다. |
