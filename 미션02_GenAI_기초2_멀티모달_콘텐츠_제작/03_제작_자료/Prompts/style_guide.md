# EchoLens Style Guide

## 1. 프로젝트 개요

| 항목 | 내용 |
|---|---|
| 브랜드명 | EchoLens |
| 제품명 | EchoLens AI Smart Glasses |
| 광고 목적 | 브랜드 인지도 향상 |
| 광고 길이 | 10초 이내 |
| 화면 비율 | 16:9 |

## 2. 브랜드 비주얼 방향

EchoLens의 비주얼은 Apple의 프리미엄 감성, GoPro의 역동적인 여행, 영화 같은 시네마틱 연출을 결합한다. 화면은 간결하고 정제된 구성을 유지하면서 세계 여행의 움직임과 현장감을 생생하게 보여준다. 제품은 여행 경험을 방해하지 않는 자연스러운 방식으로 노출하며, AI가 여행자의 시선을 기록한다는 핵심 기능을 직관적으로 전달한다.

## 3. 주인공(Character)

| 구분 | 기준 |
|---|---|
| 인물 | 20대 한국인 남성 여행 크리에이터 |
| 외형 | 자연스러운 짧은 검은 머리, 단정하고 편안한 인상 |
| 복장 | 로고가 없는 무채색 상의와 가벼운 여행용 재킷, 실용적인 여행 복장 |
| 분위기 | 꾸미지 않은 자연스러움, 여행을 즐기는 자유롭고 활기찬 태도 |
| 표현 | 과장된 포즈보다 실제 여행 중인 듯한 자연스러운 표정과 움직임 |

모든 Scene에서 동일한 인물의 얼굴, 헤어스타일, 복장과 연령대가 유지되어야 한다.

## 4. EchoLens 제품 디자인

EchoLens AI Smart Glasses는 일상적인 안경처럼 자연스럽게 착용할 수 있는 슬림한 형태로 정의한다. 프레임은 무광 블랙, 렌즈는 투명, 소재는 가볍고 매끄러운 금속과 합성 소재의 조합으로 표현한다. 안경 다리는 얇고 단정하게 유지하며, 소형 카메라와 센서는 프레임에 돌출되지 않도록 통합한다. 전체 디자인은 장식 요소를 줄인 미니멀하고 프리미엄한 인상을 유지한다.

## 5. 촬영 스타일

- **Ultra Realistic:** 인물, 제품, 여행지를 실제 촬영처럼 사실적으로 표현한다.
- **Cinematic:** 영화 같은 화면 구성과 자연스러운 심도, 부드러운 장면 전환을 적용한다.
- **Natural Lighting:** 여행지의 실제 시간대와 환경에 어울리는 자연광을 사용한다.
- **HDR:** 하늘, 인물, 풍경의 밝고 어두운 영역을 균형 있게 표현한다.
- **POV:** 여행자의 1인칭 시점을 중심으로 현장감과 몰입감을 전달한다.
- **Dynamic Movement:** 걷기와 이동 장면은 역동적으로 표현하되 화면 흔들림은 과하지 않게 유지한다.

## 6. 색감(Color Palette)

전체 색감은 프리미엄 이미지를 위한 블랙과 화이트의 절제된 무채색을 기본으로 한다. 여행 장면에서는 자연광을 살린 선명한 하늘색, 바다색, 녹색을 사용해 생동감을 더한다. 일출과 일몰 장면에는 따뜻한 골드 톤을 적용해 시네마틱한 분위기를 강조한다. 모든 장면은 과도한 채도나 인위적인 색 보정을 피하고, 자연스럽고 선명한 대비를 유지한다.

## 7. 화면 비율 및 해상도

- 화면 비율: 16:9
- 권장 해상도: 1920×1080
- 프레임레이트: 24~30fps
