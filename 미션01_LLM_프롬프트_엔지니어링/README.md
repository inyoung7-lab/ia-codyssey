# LLM Prompt Engineering Project

회의록을 요약하고 Action Items를 추출하는 LLM 프롬프트 엔지니어링 프로젝트입니다.

---

## 프로젝트 소개

- **프로젝트 목적:** LLM을 활용한 업무 자동화 프롬프트 패키지 완성
- **수행한 업무 과업:** 회의록 요약 및 Action Items 추출
- **최종 목표:** 실제 업무에 사용할 수 있는 일관된 형식의 회의록 요약과 담당자별 Action Items 생성

## 프로젝트 결과 요약

동일한 회의록과 Prompt를 사용하여 GPT-5.4 mini, Claude Haiku 4.5, Gemini 3 Flash의 결과를 비교했습니다. 세 평가 모델의 결과와 후속 검증을 바탕으로 GPT-5.4 mini를 최종 모델로 선정했습니다. Prompt v1을 역할, 출력 형식, 확인 질문, Hallucination 방지 및 문맥 유지 규칙을 포함한 Prompt v2로 개선했습니다. 최종 Prompt를 적용한 10턴 대화에서 조건 변경, 추가 정보 반영, 문맥 유지 및 사실 기반 응답을 검증했습니다.
